# **MixOS**
MixOS is a new operating system that combines intuitiveness, ease of use and extensive features.

## **Project objectives**
The project was created so that you can discover a new world of intuitiveness, simplicity and UX. Management and interface will be at the level of leading solutions in the market of desktop and mobile OS.

## **Roadmap**
The OS will be supported and updated for a long time. This section of the documentation will contain the main plans for the OS.
* **Minimum boot loader and kernel ✔️**
* **Drivers and file system**
* **System calls and user mode**
* **Built-in compiler for a custom programming language — C/**
* **GUI and graphics stack**
* **ARM support**
